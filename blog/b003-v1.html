First Website In Place

November 1, 2012

I enjoyed creating the web site. Michael Lemley, a business development manager, said, “Because of the nature of your company, the site should have publisher-editor quality.” I set up web site formatting styles and layouts to be compatible with the ebooks.

After user testing by Robin Smith, marketeer, and Homer Yau, a QA specialist, I added all the interactivity we have come to expect from web sites—click the company logo to go to the home page, drop down menus for a quick overview of content, mouse over messages for descriptions and summary information, and large graphic popups when clicking on book thumbnail images. I chose to use jQuery and JavaScript to make the web site Web 2.0 user friendly.

The home page graphics got excellent reviews after I followed Robin’s visualization suggestions. Marketing people are pros at making things look better. To add more visual eye candy, I added icons to the drop down menus.

The web site is now ready for content.
